# TransitHub 🚍

TransitHub é uma plataforma de busca de passagens desenvolvida com React e Tailwind CSS.
O objetivo do projeto é simular uma aplicação moderna de comparação de viagens, focada em praticar desenvolvimento front-end e criação de interfaces intuitivas.

---

## ✨ Sobre o projeto

O TransitHub permite ao usuário:

- Pesquisar origem e destino de viagens
- Simular busca de passagens
- Explorar uma interface moderna e responsiva
- Interagir com componentes dinâmicos (React)

O projeto está em fase inicial de desenvolvimento, com foco na construção da interface (UI/UX).

---

## 🛠️ Tecnologias utilizadas

- React
- Tailwind CSS
- JavaScript (ES6+)
- Vite
